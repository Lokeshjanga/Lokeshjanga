## 🚀 Featured projects

### [zerodha-automate](https://github.com/Lokeshjanga/zerodha-automate) — Zerodha Trade MCP Server

> A production-quality MCP server that gives AI agents live access to your Zerodha trading account

Built a **Model Context Protocol server** in TypeScript that lets any MCP-compatible AI client (Claude, etc.) interact with Zerodha's Kite Connect API. The agent can view your portfolio, check margins, and place live buy/sell orders — with an explicit `confirm: true` guard so no accidental trades fire.

**What makes it production-grade:**
- Zero hardcoded secrets — full `.env` + `.env.example` setup
- Input validation before any order execution
- Dry-run mode by default; live trade requires explicit confirmation
- TypeScript throughout with `tsconfig.json` and type-checking scripts
- Test suite with mocking (no real network calls in tests)
- MCP stdio transport — plugs directly into Claude Desktop or any MCP client

`TypeScript` `Node.js` `Bun` `MCP` `Zerodha Kite Connect API` `LLM Tool Use`
